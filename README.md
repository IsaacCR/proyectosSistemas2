# proyectosSistemas2
Desarrollo del Sistema Web para proyectos de Sistemas 2 URL

Para poder ejecutar laravel necesitan instalar lo siguiente en su computadora:
   - https://getcomposer.org/Composer-Setup.exe		Composer
   - https://desktop.github.com/			Git
   - https://www.apachefriends.org/es/index.html	Xampp (pueden usar otro si gustan)

Pueden ver una guía de instalación de Laravel en el siguiente enlace: http://tutsnare.com/how-to-install-laravel-on-windows-xampp/ (solo necesitan llegar hasta el paso #2)

Luego de tener todo lo anterior instalado deben clonar el repositorio dentro de la carpeta htdocs de xampp (o el que hayan elegido).

Al terminar de clonar el repositorio deben realizar el paso de Using Shell menus que se encuentra en la guía de instalación de Laravel (click derecho sobre el folder y seleccionar "Use Composer here", luego ejecutar el comando "composer install").

Luego de haber hecho la instalación de dependencias con composer debe estar funcionando ya la aplicación, para verificar si todo está bien deben de iniciar el servicio de Apache en Xampp y en el explorador escribir la dirección hacia el directorio public de la instalación (si no realizaron ninguna configuración especial en Xampp la ruta del explorador debe ser algo como "http://localhost/proyectosSistemas2/public/"), y les debe de mostar una página con el texto Laravel 5.
